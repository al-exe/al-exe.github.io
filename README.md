## Description
Colony Clicker is a React web application used to assist in MMEJ (microhomology-mediated end joining) research. It was built as a successor to [Colony Counter](https://github.com/al-exe/colony-counter), which ambitiously attempted to automate the counting process entirely.

## Basic usage
1. Go here: [Colony Clicker](https://al-exe.github.io/).
2. Upload a `.png` or `.jpg` file for labeling.
3. Click anywhere within the image to create a label.
4. Use the controls to pan, zoom, and adjust label settings.
5. Download your finished work or discard the image to start a new one.

#### Usage tips
* To undo your last created label, press `ctrl + z` or `command + z`.
* Adjust your label settings (label type, color, size) in the right hand side.
* Use the on-canvas `+` and `-` buttons to zoom, and drag to pan.
* Use the reset button to return to the fit-to-container view.

## Data policy
Images are stored locally in your browser to preserve your session after refresh. No user or usage data is collected, and images are not uploaded to any server.

## Limitations
Browser zoom can cause alignment issues; use the app zoom controls for best accuracy.
