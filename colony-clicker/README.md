# Colony Clicker

Image labeling tool for quick counting and annotation. Upload an image, click to
drop numbered markers or dots, and export the result.

## Features

- Click to place numbered labels or dots
- Adjustable label size and color
- Undo last label (Cmd/Ctrl+Z)
- Clear labels or discard image
- Pan/zoom with on-canvas controls
- Export as PNG (Download image)
- Session persistence via localStorage

## Usage

1) Upload a PNG or JPEG.
2) Click on the image to place labels.
3) Use the right sidebar to change label type, color, and size.
4) Download the labeled image when finished.

## Controls

- Add label: click on the image
- Undo: Cmd/Ctrl+Z
- Pan: click + drag
- Zoom: use the + and - buttons (top-right of image)
- Reset view: circular arrow button
